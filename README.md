# Weather App

A responsive weather application built using HTML, CSS, JavaScript, and WeatherAPI.

## Live Demo

[Live Demo] https://yashmangal72.github.io/Weather-App/

## Features

- Search weather by city name
- Real-time weather data using WeatherAPI
- Dynamic weather icons
- Temperature display in Celsius
- Weather condition display
- Loading state while fetching data
- Error handling for invalid city names
- Responsive design
- Modern glassmorphism UI
- Weather-based Background Changes
- Many Weather Parameters

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6+)
- Fetch API
- Async/Await
- WeatherAPI

## Installation

1. Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Weather-App.git
```

2. Open the project folder

```bash
cd Weather-App
```

3. Open `index.html` in your browser
